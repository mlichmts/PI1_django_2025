10.css, obrázky
-Vytvorit v priecinku aplikacie preičinok static/názov_aplikácie
-v tomto priecinku vytvorit css(naprikal kalkulacka.css)
-do tohto pirecinku nakopirovat obrazky
-do template(index.html) doplnit:
    -uplne na zaciatok doplnit{% load static %}
    -do hlavicky odkaz na style
    <head>
    <link rel= "stylesheet" href="{% static 'kalkulacka/kalkulacka.css' %}">

11. Modely(models.py)

from django.db import models


class Osoba(models.Model):
    meno = models.CharField(max_length=100)
    priezvisko = models.CharField(max_length=100)
    vek = models.IntegerField()
    datum = models.DateField()
    email = models.EmailField()
    muz = models.BooleanField()

    def __str__(self):
        return f"{self.meno} {self.priezvisko}"
